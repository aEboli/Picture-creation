# 版本说明-v0.6.0

对应对象：Commerce Image Studio

发布日期：2026-03-16

## 本版重点

- 参考图复刻模式升级为受约束复刻工作流
- 创作台上传区、参数区、数量区进一步压缩和对齐
- 飞书字段模板更新为更稳定的字段顺序
- 图片预处理链路改为更高保真策略
- 默认数据目录与其它项目隔离

## 核心更新

1. 参考图复刻支持复刻目标、构图锁定、文案区域策略和 1 到 4 个复刻变体
2. 上传区改为预览框内上传与拖拽替换
3. 设置页提供新旧两套飞书映射模板
4. 原图和生成图同步链路继续兼容旧缓存文件

## 产物

- 安装器：`IMAGE-STUDIO-WINDOWS-0.6.0.exe`
- 发布标签：`v0.6.0`
